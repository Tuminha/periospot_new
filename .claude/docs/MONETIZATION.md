## API Integrations (Automated Link Creation)

### Genius Link API

Creates geo-targeted affiliate links that automatically redirect users to their local Amazon store.

**Configuration:**
```
API Key: 847ce64e10274a2ebb731e9f2e6e44c5
API Secret: abbfaa86d9f04c9a85c71d26210da56a
Endpoint: https://api.geni.us/
Default Group ID: 416233 (PerioSpot)
```

**Usage:**
```bash
# Create link from URL
node scripts/geniuslink-api.cjs create "https://amazon.com/dp/ASIN"

# Create link from ASIN
node scripts/geniuslink-api.cjs asin B08N5WRWNW

# Result: https://geni.us/CODE
```

### Tana API

Saves affiliate links to Tana with #affiliate-links supertag for tracking.

**Configuration:**
```
Token: eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...
Endpoint: europe-west1-tagr-prod.cloudfunctions.net/addToNodeV2
```

**Usage:**
```bash
# Add affiliate link to Tana
node scripts/tana-api.cjs add "Product Name" "https://geni.us/xxx" amazon books

# Test connection
node scripts/tana-api.cjs test
```

**Note:** The Tana Input API is write-only. To read existing affiliate links, set up the [supertag-cli MCP](https://github.com/jcfischer/supertag-cli).

### Combined Affiliate Manager

All-in-one tool for creating and tracking affiliate links:

```bash

node scripts/affiliate-manager.cjs create-link "Product Name" "https://amazon.com/dp/XXX" books

# Quick product links (pre-defined)

node scripts/affiliate-manager.cjs quick lindhe-periodontology

# Generate markdown for blog posts
node scripts/affiliate-manager.cjs markdown "https://geni.us/xxx" "Product Name" books
```
